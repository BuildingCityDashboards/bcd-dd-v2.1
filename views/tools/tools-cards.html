Planning Applications

Explore planning applications across the four local authorities. Planning applications are located on a map and can be filtered in various ways, including by date range, size of development and application status. You can also search directly for an application number.

Geodemographics

Geodemographic classification is a tool that groups spatial units based on socio-economic characteristics. The Dublin Geodemographic Tool is following the Ireland Geodemographic classification method but focusing on Dublin for a better insight. For the Dublin Geodemographic Tool, Small Areas are selected as spatial units and 2016 Census of Population results are used for socio-economic components.
