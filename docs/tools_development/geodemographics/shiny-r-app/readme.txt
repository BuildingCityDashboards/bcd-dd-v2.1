Dublin Clusters:

“Dublin_clusters.csv”
The last column of this file is the cluster category for each small area in Dublin. The column is titled as ‘cluster’.

“Dublin_zscores.csv”
These are the values for the 33 variables used in the study for each cluster. This file is the input for the plot presented on the interface.

Interface :

This interface is produced using R Leaflet Shiny integration. I am sending the code “Dublin_viz” as well as the relevant files “n.RData’ and ‘zv.Rdata’ for this code to run. To give you an idea how the interface looks like, some screenshots are added to the folder.
